
# React Projects Workspace

This workspace contains several web app projects built with React, Next.js, Vite, and Flask. Below is a summary of each project:

## 1. firststack (Book Storage App)
- **Frontend:** React + Vite
- **Backend:** Flask (SQL database)
- **Description:** Users can add and manage their book collection. Flask stores book data (title, author, description, etc.) in a SQL database, and React displays the collection.

## 2. joku (Blog App)
- **Frontend:** Next.js (React)
- **Backend:** Flask (serves blog data as JSON)
- **Description:** A blog platform where users can write and view blogs. The frontend fetches blog data from a Flask backend.

## 3. todo (Basic Todo App)
- **Frontend:** Next.js (React)
- **Description:** A simple todo list app for managing tasks.
